# skill-deployer

> 将任何本地 skill 部署为独立 GitHub 仓库。自动生成 README、LICENSE(MIT)、版本标签、GitHub Actions 验证。

## 安装
```bash
npx -y skills add songxrui/skill-deployer -g --all
```

## 触发方式
| 触发词 | 示例 |
|--------|------|
| 部署skill | "把这个skill部署到GitHub" |
| skill发布 | "通过skill-review门禁后发布" |
| 创建skill仓库 | "批量部署这几个skill" |

## 质量保证
| 指标 | 值 |
|------|-----|
| 设计标准 | 收费产品级 (G1-G6 全过) |
| Token 预算 | ~2900 tokens |
